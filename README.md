# 🚗 Electric Vehicle Dashboard – Tableau

![Dashboard Preview](ev_dashboard.png)

## Overview
An interactive Tableau dashboard analysing **150,480 electric vehicles** registered across the United States. Built to explore EV adoption trends, manufacturer dominance, and regional distribution.

## Key Insights
- **150.48K total EVs** analysed with an average electric range of **67.88 km**
- **Tesla dominates** with 68,983 vehicles — nearly 5x the next competitor (Nissan at 13,497)
- **BEV vehicles make up 77.62%** of total, with PHEV at 22.38%
- EV registrations **peaked in 2022** at 37.08K before dropping in 2023
- **Top models:** Tesla Model Y (18.94%), Model 3 (18.41%), Nissan Leaf (8.76%)
- **62.95K vehicles** eligible for Clean Alternative Fuel Vehicle (CAFV) programme

## Visualisations
| Chart | Description |
|-------|-------------|
| Area Chart | EV registrations by model year (2012–2023) |
| Bar Chart | Top 20 vehicle makes by count |
| Donut Chart | CAFV eligibility breakdown |
| Map | Vehicle make distribution by US state |
| Table | Detailed breakdown by model, make, and EV type |

## Filters
- Clean Alternative Fuel Vehicle type
- Electric Vehicle Type (BEV / PHEV)
- Model, Make, and State

## Tools Used
- **Tableau** – dashboard design and visualisation
- **Dataset** – US Electric Vehicle Population Data (Washington State)

## How to View
1. Download `vehicle1.twb` from this repository
2. Open in **Tableau Desktop** or **Tableau Public**
3. Use the filter panel on the left to explore by state, make, or model
